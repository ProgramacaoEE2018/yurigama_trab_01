# yurigama_trab_01
Programa feito em Ling C, utilizando CodeBlocks.
Usúarios: alunos CG/CFG do IME.
Finalidade: calcular a nota mínima necessária na VF para aprovação em cada disciplina, a partir da inserção das notas de VE e VC.
Funcionamento: usuário insere a nota de média de VE e a nota de VC obtidas em determinada disciplina e o programa exibirá na tela a nota mínima na VF para aprovação nessa disciplina.
